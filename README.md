# HEWS - Hackernews

Hacker news client

## Setup
1. Clone the repository 
2. run `yarn` in the root dirctory

## Running
`yarn dev`

## Running tests
`yarn test`

## Production build
`yarn build`

## Tech Stack
1. React
2. TypeScript
3. Vite
4. React Router
5. React Query
6. Zag UI + Tailwind

## Functions supported
1. Search
2. HN Front Page
3. Ask HN
4. Show HN
5. Pagination
